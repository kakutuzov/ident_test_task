Мое решение тестового задания.

Структура:

 -- Ответы на задание 1 и 2 хранятся в папке "DataBase". Для их решения я решил использовать базу Postgres,
    т.к в России она набирает особую популярность, а я ранее мало про неё что знал. Совместил приятное с полезным.
    Сама база разворачивется в докер контейнер, к которому я в последствии подключался через DBeaver. Для удобства
    Развертывания так же сделал docer-compose файл, что даёт возможность запустить БД двумя командами: docker-compose build и
    docker-compose run. Все запросы хранятся в DataBase\queries. В папке DataBase\explanations лежит моё пояснение к заданю 1
    (схема и 3 текстовых файла с кратким пояснением). С заданием 2 я не справился, на мой взгляд... В какой-то момент упёрся
    и потратив много времени решил отложить на потом, но потом потравтив ещё кучу времени на остальные задания понял, что времени
    у меня не осталось (со слов автора задания среднее выполнение 6 часов, я потратил где-то часов 12).

-- Ответ на задание 3 хранится в DotNet\WhatsWrong\WhatsWrong.cs пояснение написал прямо там в виде комментария.   	

-- Решение для 4 задачи хранится в DotNet\FindThem. В корне DotNet есть файл EntryPoint.cs внутри которого распологаются вызовы
   все релизованных методов для поиска, в результате выполнения программы в консоль будет выведен небольшой отчет о работе
   каждого из 3 методов поиска.

-- Решение для 5 задания хранится в папке JS. Точкой входа является файл index.html. В условии задания было требование на отправку POST
   запроса, но я решил сдлеть GET запрос, надеюсь это не повлияет на оценку.

-- Решение для 6 задания хранится в папке ASP.

   местах работы. Тем не менее у меня есть аккаунт на GitHub и на GitLab, в котором можно найти небольшые проекты. Все они являются лишь моими
